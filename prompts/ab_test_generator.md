# Prompt: A/B/C TikTok Script Generator

Create three 15-second US TikTok Shop UGC variants for one product. Use only facts marked usable in the fact ledger.

- Version A: discovery hook, direct proof, low-pressure CTA.
- Version B: relatable problem, casual product introduction, low-pressure CTA.
- Version C: immediate visual demonstration, close-up proof, low-pressure CTA.

For each version return:

- `0-2s` hook with the product visible
- `2-5s` context
- `5-10s` safe product demonstration
- `10-13s` believable payoff
- `13-15s` CTA
- exact voiceover
- safe-center captions
- visual direction
- fact IDs used

Target 30-45 spoken English words and natural American phrasing.
